# iOS-style React Calculator

This is a premium, iOS-style calculator built with React, Vite, and TailwindCSS.

## Features
- **Authentic iOS Design**: Dark mode, circular buttons, active states, and typography.
- **Responsive**: Works on Mobile, Tablet (scaled), and Desktop.
- **Full Keyboard Support**: Type numbers, operations, Enter/=, Backspace, Escape/AC.
- **Smart Logic**: Chain calculations, handle floating point errors, and formatted display.

## How to Run

1.  **Install Dependencies** (if not already done):
    ```bash
    npm install
    ```

2.  **Start Dev Server**:
    ```bash
    npm run dev
    ```
    Then open the URL shown (usually `http://localhost:5173`).

3.  **Build for Production**:
    ```bash
    npm run build
    ```
